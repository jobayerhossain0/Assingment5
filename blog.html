Important FAQ

What is Document Object Model (DOM)?

The html DOM is a standard object model and programming interface for HTML. It defines the HTML elements as objects The properties of all HTML elements The methods to access all HTML elements The events for all HTML elements

How do you select an element from the DOM?

I can select an element from the dom by defining an id on the element and document.getElementById method

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the context of the DOM where a single event listener is attached to a parent element to handle events for its child elements, rather than adding multiple event listeners to individual child elements. It is useful because of improved performance, dynamic content handling, and simplified code.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an elements attributes with methods like getAttribute(), setAttribute(), or by directly modifying the property and styles using the style property or by adding or removing CSS classes with classList.
